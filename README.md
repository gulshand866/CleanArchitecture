# Clean Architecture !

**This project is used to demonstrate the use of Clean Architecture in various Microservices**

There are four microservices and a gateway :-

1. CleanProductAPI
2. CleanUserAPI
3. CleanCartAPI
4. CleanOrderAPI

&

5. CleanGateway

All the Microservices follow "Clean Architecture"

**Tech Stack**

- [ ] .NET 8 - Backend
- [ ] Kafka - For Inter-Services Communication
- [ ] There is no actual database used - Different DS has been used for in-memory data storage
